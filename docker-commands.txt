to build a docker img --> docker build -t mlops-docker-demo <name of the img>
to run a docker img --> docker run -p 5000:5000 mlops-docker-demo <name of the img>
to tag a docker img --> docker tag mlops-docker-demo<img name> lakshmanbunny<docker hub username>/mlops-docker-demo<img name>:latest<tag>
to push a docker img to docker hub repo --> docker push <username>/<img name>:<tag>
to pull img from dockerhub --> docker pull <username>/<img name>:<tag>
to run the pull image --> docker run -p 5000:5000 lakshmanbunny/mlops-docker-demo:latest<tag>
